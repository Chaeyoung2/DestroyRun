1. 이름(학번) : 손채영(2015182016)
2. 게임 개요 : 눈 앞에 방해되는 오브젝트들을 박살내며 달리는 러닝 게임
3. 개발에 사용한 외부 라이브러리 리스트 및 소개
4. 자체 제작 에셋 리스트 및 소개
5. 개발 내용 중 강의를 통해 학습한 부분이 어떻게 활용되었는지
 클래스 상속 : 체력, 죽음 상태를 공통으로 갖고 있는 클래스는 LivingObject 클래스를 상속 받는다.
 get, set property : 각 클래스의 체력, 죽음 상태 얻어온다.
 싱글톤 패턴 : 점수 정보를 가지고 있는 등 포괄적 관리를 하고 있는 GameManager 클래스
 씬 관리 : Menu, Main 씬
 Rigidbody를 이용한 플레이어 이동
 UI - TextMeshPro를 이용한 한글 머태리얼 생성, Image Component, InputComponent, button Animation
